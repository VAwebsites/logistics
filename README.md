## Logistics Management

This powerful CRM was developed using Laravel, Vuejs and Laravel API.

Nodejs version
12.0.0

### Features

+ SPA
+ Customizable
+ Create Shipment
+ Print Invoice
+ Send quotes
+ Download Reports
+ Customer login
+ Staff Login
+ Send SMS
+ Send Email
+ Receive quotes
+ Send Invoices
+ Beautiful mail templates
+ Expense Manager
+ Vendor Management
